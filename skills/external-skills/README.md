# external-skills/

Skills provenant d'ailleurs — marketplace Claude Code, autre équipe, partagé par quelqu'un. Rien ici n'est développé par nous.

## Règle

**Ne jamais éditer un skill externe en place.** Si un ajustement semble nécessaire :
- Soit il est mineur et générique → proposer la modification en amont (auprès de la source).
- Soit il est spécifique à notre usage → c'est un signe que le skill devrait être copié dans `../purk-skills/` sous un nouveau nom, pas modifié ici. À partir de ce moment, c'est le nôtre, avec son propre suivi.

## Comment en ajouter un

Voir la section « Ajouter un skill externe » dans le `README.md` à la racine du repo (copie figée vs git submodule).
